img标签

这是图片标签，src属性是图片的来源，alt是对图片的描述，当图片显示不出来的时候，这里的描述就会显示出来,width可以设置图片的宽度，是成比例放大缩小的，border是图片外部的边框，style就是样式。border-color：设置颜色。

<img src="1.png" alt="这是一张图片" width="300px" border="2px" style="border-color: red"/>

ul,ol标签

列表标签


<!--有序列表和无序列表-->
<ul type="square"><!--type 属性可以取disc(实心圆),circle(空心圆),square(方块)-->
    <li>英雄</li>
    <li>英雄</li>
    <li>英雄</li>
</ul>
<ol type="i" start="2"><!--type属性用于如何显示（可以取1,A,a,i,I） statrt属性是从第几个开始显示 ol orderlist有顺序的列表-->
    <li>英雄</li>
    <li>英雄</li>
    <li>英雄</li>
</ol>

framest框架

主要就是用于对页面的分割，但这里面不能有body标签

cols=“400px,*”(按像素分) cols=“50%,*”(按百分比分)

target：还可以取_black（打开全新页面），_self（替换本页面）默认是 _self

html>
<!--noresize 不允许各窗口改变大小 frameborder边框大小 scrolling是否有滚动条 name 给frame取名-->
<frameset cols="50%,*">
    <frame src="xiao.html" noresize scrolling="no" frameborder="0px"/>
    <frame src="login.html" name="login"
    frameborder="0px"/>
</frameset>
</html>

<!--xiao.html-->
<!DOCTYPE HTML>
<html>
<head>
    <meta charset="utf-8">
</head>
<!-- target 表示我们点击目标后，目标指向谁，主要这里就是写的框架的name-->
<body>
    <a href="1.html" target="login">啦啦啦</a><br/>
    <a href="login.html" target="login">呀呀呀</a><br/>
</body>
</html>
